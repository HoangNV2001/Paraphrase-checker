FAQs

How to use?

Step 1: Enter the document that you want to check by filling in the text field, or by inserting a Pdf or Txt file.
Step 2: Enter the keywords for the document.
Step 3: Select the number of search results that you want the engine to search and compare with your document.
Step 4: Press 'Start search' button, and the system will handle the rest.

What happens after I press the 'Start search' button?

  • Through Google search API, the system would search for Pdf files using the keywords and the number of search results that was given by the user.
  • The files found in the search result would be downloaded and preprocessed for comparison.
  • Each sentence from the input document will be examined to determine whether it is a paraphrase of any sentence in the referential documents.
  • Redirect the user to result page. Delete all the downloaded files, as well as the input document.

    Notice: Since all temporary files are deleted after comparison stage for privacy purpose, the result page can only be loaded once.
    Attempting to reload the result page would trigger redirection to the main page.
